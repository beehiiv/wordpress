Official beehiiv WordPress plugin.


== Description ==

Connect WordPress with beehiiv to publish posts as newsletters.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/beehiiv` directory, or install through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin in the WordPress admin area.


== Frequently Asked Questions ==

= Does this require a beehiiv account? =

Yes. You will need an active beehiiv account.
